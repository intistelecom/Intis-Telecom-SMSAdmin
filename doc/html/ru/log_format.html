Формат лога¶

Лог файл представлен ввиде файла с разделителями (csv, разделитель |). Каждая новая запись лога отделена от другой переносом строки (\n)

Описание формата и назначения колонок¶

Назначение Формат Допустимые значения
Время строка, ISO8601 Цифры, -, :, Т, +
Уровень лога строка debug, error, info, warn
Модуль программы строка Текст, имя модуля программы
Текст строка Любые текстовые символы, кроме переноса строки
