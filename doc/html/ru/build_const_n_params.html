Константы и параметры сборки¶

Параметры сборки задаются при сборке и хранятся в файле include/global.h

Параметры сборки¶

Имя параметра Описание Значение по-умолчанию Редактируемый пользователем Перезаписываемый в процессе выполнения сценария Пример
SMSADMIN_BUILD_DATE Дата сборки программы. Время в формате ISO8601 Определяется при сборке из системного времени Нет Нет 2014-07-01T14:01:45+0400
SMSADMIN_VER_STRING Номер версии прораммы major.minor.patch Нет Нет 0.1.1
SMSADMIN_VER_NAME Имя версии программы Определяется разработчиком Нет Нет Cold Start
SMSADMIN_DEFAULT_CONF_FILE Имя файла конфига по-умолчанию /etc/smsadmin.cfn Да Да, Параметр командной строки -с /etc/smsadmin.cfn
SMSADMIN_DEFAULT_LOG_FILE Имя файла лога по-умолчанию /var/log/smsadmin.log Да Да, Параметр командной сроки -l, Параметр файла конфигурации log /var/log/smsadmin.log
SMSADMIN_MAKE_SHARED Статически линковать библиотеки boost, sms16xapi, системные: libstdc++, libgcc OFF Да Нет cmake -DSMSADMIN_MAKE_SHARED=ON
SMSADMIN_SECURITY_PURIFY_DISABLE Отключить скрытие аторизационных данных в логе OFF Да Нет cmake -DSMSADMIN_SECURITY_PURIFY_DISABLE=ON
